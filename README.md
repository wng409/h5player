🎬 HTML5 视频播放器

一个功能丰富、界面美观的现代化 HTML5 视频播放器，支持本地视频播放和在线视频流媒体播放。

https://via.placeholder.com/800x400/1a2a6c/ffffff?text=HTML5+Video+Player

✨ 主要功能

🎯 核心特性

· 双模式播放：支持离线本地视频和在线视频播放
· 响应式设计：完美适配桌面端和移动端
· 高清播放：支持各种分辨率的视频播放
· 完整控制：播放、暂停、进度调节、音量控制
· 全屏模式：提供沉浸式观看体验

📁 离线播放模式

· 本地视频文件导入
· 支持 MP4、WebM、OGV 等主流格式
· 文件信息显示（名称、大小、格式、分辨率）
· 最近播放历史记录

🌐 在线播放模式

· 支持网络视频 URL 播放
· 自动识别视频格式
· 快速加载和缓冲

🚀 快速开始

方法一：直接使用

1. 下载 index.html 文件
2. 用浏览器打开即可使用

方法二：部署到 GitHub Pages

1. Fork 或下载此项目
2. 上传到 GitHub 仓库
3. 在仓库设置中启用 GitHub Pages
4. 访问您的在线播放器

📖 使用方法

离线播放

1. 点击"离线播放"模式
2. 点击"选择视频文件"按钮
3. 选择本地视频文件
4. 开始播放

在线播放

1. 点击"在线播放"模式
2. 输入视频URL地址
3. 点击"加载在线视频"
4. 开始播放

🔧 自定义配置

修改主题色彩

在 CSS 中修改以下变量：

```css
/* 主色调 */
background: linear-gradient(135deg, #1a2a6c, #b21f1f, #fdbb2d);
/* 按钮颜色 */
background: #ff5722;
```

添加新功能

在 JavaScript 中添加新的控制功能：

```javascript

speedBtn.textContent = '1x';

    // 切换播放速度逻辑

```
